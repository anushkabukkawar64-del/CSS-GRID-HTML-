1. Where does gap add space?
Gap adds space between grid items (rows and columns).
2. How is gap different from margin?
Gap adds space inside the grid between items. Margin adds space outside the element.
3. Does gap affect container size?
Yes, gap space is included inside the grid container layout.
4. Can we use gap only for rows or columns?
Yes, we can use row-gap or column-gap separately.
5. Why is gap preferred in Grid layouts?
It gives clean spacing without adding margins to each item.
