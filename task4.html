1. What does auto do in grid rows?
auto allows the row to grow based on the content inside it.
2. Which row grows based on content?
The row that contains more content expands automatically.
3. Can rows have different heights?
Yes, grid rows can have different heights using px, fr, %, or auto.
4. What happens if content is very large?
The row expands to fit the content unless a fixed height forces overflow.
5. Where are grid rows useful in real layouts?
Useful in dashboards, forms, admin panels & structured page layouts.
