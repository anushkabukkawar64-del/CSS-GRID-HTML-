Task 8: Auto-Fit, Minmax & Responsive Grid — Answers

1. What problem does auto-fit solve?
auto-fit fills the row with as many items as possible and moves the rest to the next row automatically.
2. Why is minmax important?
minmax sets a minimum width so items don’t shrink too small and a maximum width so they can expand.
3. Do items shrink too much?
No, minmax prevents extreme shrinking by enforcing a minimum size.
4. Is this layout mobile friendly?
Yes, number of columns reduces on small screens without breaking.
5. Why are media queries not needed here?
Because responsive behavior is handled automatically through minmax and auto-fit.
