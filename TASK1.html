Task 1: CSS Grid Introduction

Grid Container With Answers:

1. Difference between normal div and grid container:
A normal div puts items vertically. Grid can arrange items in rows & columns.
2. Why layout looks same after applying Grid?
Because Grid needs columns or rows to change layout.
3. Are grid items created automatically?
Yes, every direct child inside a grid container becomes a grid item.
4. Can Grid work without rows or columns?
Yes, but visually nothing changes until rows/columns are defined.
5. What does Grid need to show visible changes?
It needs grid-template-columns, grid-template-rows & gap.
