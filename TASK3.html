Task 3: Fixed vs Flexible Columns

Grid Container With Answers:

1. Which layout breaks faster on small screens?
Fixed px layout breaks faster because columns cannot shrink.
2. Which layout adjusts automatically?
Flexible fr layout adjusts because fr shares available space.
3. Why is fixed layout risky?
Fixed px does not respond to different screen sizes and can overflow.
4. Where can fixed columns be useful?
Useful for structures needing exact sizes like sidebars or images.
5. Which unit would you choose for a responsive website?
fr, because it makes columns resize automatically.
