Task 6: Grid Item – Column Span — Answers

1. What are grid lines?
Grid lines are the numbered lines that define rows and columns in a grid layout.
2. How many columns does the item span?
The item spans multiple columns depending on the start/end grid line values.
3. What happens to other items?
Other items shift to the next available grid spaces without overlapping.
4. Can two items overlap?
Yes, but only if overlap is defined manually using positioning.
5. Where is column spanning used in websites?
Used in dashboards, headers, galleries, hero sections & layouts needing wider content.
